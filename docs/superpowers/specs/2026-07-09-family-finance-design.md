# Family Finance Inc. - 家庭财务公司化管理系统

## 一、项目定位与核心价值

### 1.1 项目概述

将企业财务管理的专业方法引入家庭，通过三大财务报表（资产负债表、利润表、现金流量表）帮助家庭清晰了解财务状况，同时支持家庭成员协作管理。

### 1.2 核心理念

- **公司化思维**：把家庭当作一个公司来经营和管理
- **专业财务视角**：用企业级财务报表思维管理家庭财务
- **智能协作**：通过大模型对话和图片识别实现高效数据录入和分析

### 1.3 核心价值

| 价值维度 | 具体内容 |
|---------|---------|
| 专业财务视角 | 三大报表全面反映家庭财务健康状况 |
| 智能数据录入 | 大模型对话 + 图片OCR快速录入 |
| 家庭成员协作 | 多人共享、权限管理、文件协作 |
| 可视化分析 | 图表直观展示财务趋势和分布 |

## 二、三大财务报表的家庭化映射

### 2.1 映射逻辑

| 报表类型 | 企业视角 | 家庭视角 | 核心公式 |
|---------|---------|---------|---------|
| **资产负债表** | 企业家底 | 家庭家底 | 总资产 = 总负债 + 净资产 |
| **利润表** | 经营业绩 | 家庭收支 | 总收入 - 总支出 = 结余 |
| **现金流量表** | 现金流状况 | 家庭现金流 | 经营现金流 + 投资现金流 + 筹资现金流 = 净现金流 |

### 2.2 资产负债表映射

**资产**：
- **流动资产**：现金、银行存款、短期投资（1年以内）、应收账款（他人欠家庭的钱）
- **非流动资产**：房产、车辆、长期投资、家具家电等固定资产
- **投资资产**（按类型细分）：
  - 股票：上市公司股票、基金份额
  - 长期国债：国债、债券基金、固定收益产品
  - 黄金：实物黄金、黄金ETF、黄金期货
  - 现金：现金、活期存款、货币基金
  - 其他：理财产品、私募股权、收藏品等

**负债**：
- 流动负债：信用卡欠款、短期借款、应付账款（家庭欠他人的钱）
- 非流动负债：房贷、车贷、长期借款

**净资产**：总资产 - 总负债

**投资配置比例**：各投资类型占投资资产总额的百分比，实时计算并可视化展示

### 2.3 利润表映射

**收入**：
- 工资收入
- 投资收益（利息、股息、租金等）
- 副业收入
- 其他收入

**支出**：
- 固定支出：房贷/房租、车贷、保险、水电煤、通讯费
- 可变支出：餐饮、购物、娱乐、医疗、教育
- 投资支出：购买资产、投资本金投入

**结余**：总收入 - 总支出

### 2.4 现金流量表映射

**经营活动现金流**：
- 流入：工资、日常收入
- 流出：日常开销、房贷/房租、水电煤等

**投资活动现金流**：
- 流入：出售资产、投资收益
- 流出：购买房产、车辆、投资等

**筹资活动现金流**：
- 流入：借款、贷款
- 流出：还款、利息支付

## 四、核心功能模块

### 4.1 家庭管理模块

**功能列表**：
- 创建/编辑家庭档案
- 邀请家庭成员（邮箱/手机号邀请）
- 角色权限管理：
  - 管理员：完整权限
  - 普通成员：录入数据、查看报表
  - 只读成员：仅查看报表
- 家庭基本信息管理（名称、成员简介等）

### 4.2 数据录入模块

**功能列表**：
- **手动录入**：表单填写收入、支出、资产、负债
- **智能录入**：通过大模型对话输入自然语言（如"我今天发工资10000元"）
- **图片识别**：上传消费截图、收据发票，自动识别并录入
- **重复录入检测**：录入前自动检测是否为重复条目，提示用户确认

**重复检测机制**：
- **内容相似度检测**：基于金额、日期、描述、分类等字段计算相似度
  - 金额相同 + 日期相同 + 描述相似 → 高置信度重复
  - 金额相近 + 日期相近 + 分类相同 → 可能重复
- **图片哈希检测**：上传图片时计算感知哈希值（pHash），与历史图片对比
- **重复提示**：检测到可能重复时，展示相似条目，让用户选择是否继续录入
- **阈值可调**：重复检测的相似度阈值可配置

**智能录入示例**：
```
用户：我今天发工资10000元，其中公积金2000元
系统：已记录收入 - 工资收入：10000元（公积金：2000元）

用户：今天在超市花了350元买日用品
系统：已记录支出 - 日常开销：350元（分类：日用品）

用户：昨天超市花了350元
系统：检测到可能重复的条目：
      - 2026-07-08 日常开销 350元 日用品
      请问这是同一条记录吗？[是，跳过 / 否，继续录入]
```

### 4.3 报表生成模块

**功能列表**：
- **资产负债表**：实时计算家庭总资产、总负债、净资产，饼图展示资产分布
- **利润表**：按时间周期（月/季/年）统计收入、支出、结余，柱状图对比
- **现金流量表**：分类统计经营/投资/筹资活动现金流，折线图趋势
- **投资配置分析**：展示投资资产配置比例（股票、长期国债、黄金、现金、其他），环形图可视化
- **自定义报表**：支持按时间范围、分类维度筛选

**动态仪表板**：
- 所有报表数据实时更新，数据变化后自动刷新
- 支持自动刷新和手动刷新两种模式
- 投资配置比例实时计算并展示
- 关键指标卡片实时显示（总资产、净资产、本月结余、投资组合总价值）

### 4.4 文件管理模块

**功能列表**：
- 上传/下载文件（财务凭证、合同、收据等）
- 文件分类管理（按类型、时间、项目）
- 权限控制（谁可以查看/编辑）
- 文件预览和搜索

### 4.5 AI 对话模块

**功能列表**：
- 自然语言查询财务数据
- 财务分析与建议（如"分析一下我们家这个月的支出是否合理？"）
- 图片 OCR 识别（消费截图、收据发票）
- 智能问答（如"我们家的净资产是多少？"）

## 五、数据库设计

### 5.1 核心数据表

| 表名 | 说明 | 关键字段 |
|-----|------|---------|
| users | 用户信息 | id, email, password_hash, name |

| family_members | 家庭成员关系 | id, family_id, user_id, role |

| liabilities | 负债记录 | id, family_id, user_id, type, name, amount, created_at |

| expenses | 支出记录 | id, family_id, user_id, type, amount, description, date |
| files | 文件记录 | id, family_id, user_id, name, path, type, phash, uploaded_at |
| ai_conversations | AI对话记录 | id, family_id, user_id, content, response, created_at |

### 5.2 数据模型关系

```
users ───< family_members >─── families
users ───> assets, liabilities, incomes, expenses, files, ai_conversations
families ───> assets, liabilities, incomes, expenses, files
```

## 八、安全与隐私

### 8.1 数据安全

- 密码使用 bcrypt 加密存储
- 敏感数据传输使用 HTTPS
- 数据库字段加密（身份证、银行卡等）
- JWT 令牌有效期管理

### 8.2 权限控制

- RBAC 角色权限模型
- 数据隔离（用户只能访问所属家庭数据）
- 文件访问权限控制

### 8.3 数据备份

- 定期数据库备份
- 文件存储多副本策略
- 灾难恢复计划

## 九、部署方案

### 9.1 开发环境

## 十、风险与应对

| 风险 | 影响 | 应对措施 |
|-----|------|---------|
| AI 服务费用 | 成本不确定性 | 限额控制、缓存常用查询结果 |
| 数据安全 | 财务数据泄露 | 加密存储、权限隔离、定期审计 |
| 图片识别准确率 | OCR 识别错误 | 人工审核机制、持续优化模型 |
| 系统性能 | 报表计算慢 | Redis缓存、异步计算、数据库优化 |
